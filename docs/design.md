## 2. Визуальная система

### 2.1 Общий стиль

**Liquid Glass / Glassmorphism** на светлом фоне. Вдохновение: Apple Vision Pro light mode, macOS Tahoe Liquid Glass.

### 2.2 Фон и блобы

- **Фон:** `#f5f4ff` (очень светлый лавандовый)
- **4 анимированных CSS-блоба** — большие круги с `filter: blur(120px)`, дрейфуют через `@keyframes` (период 8–14 с, разные тайминги)
- **Glow эффект:** каждый блоб имеет `box-shadow: 0 0 120px 60px <цвет>`

| Блоб | Цвет | Glow |
|------|------|------|
| 1 | `#c9b3ff` (лаванда) | `rgba(180,130,255,0.5)` |
| 2 | `#9ff5e0` (мята) | `rgba(100,230,200,0.4)` |
| 3 | `#ffb8d9` (розовый) | `rgba(255,150,200,0.35)` |
| 4 | `#b3d8ff` (голубой) | `rgba(120,180,255,0.4)` |

Блобы расставлены асимметрично (`position: fixed`, `z-index: 0`), не скроллятся вместе со страницей.

### 2.3 Стеклянные карточки

```css
background: rgba(255, 255, 255, 0.55);
backdrop-filter: blur(28px);
-webkit-backdrop-filter: blur(28px);
border: 1px solid rgba(255, 255, 255, 0.85);
border-radius: 20px;
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.08);
```

**Hover-состояние:**
```css
border-color: rgba(255, 255, 255, 1);
box-shadow: 0 8px 40px rgba(0, 0, 0, 0.12), 0 0 20px rgba(160, 100, 255, 0.15);
transition: all 0.3s ease;
```

**macOS-chrome декор** (на Hero и Journey карточках): три точки `●●●` красный `#ff5f57` / жёлтый `#febc2e` / зелёный `#28c840`, размер 12px, в верхнем левом углу карточки.

### 2.4 Типографика

- **Шрифт:** Inter (Google Fonts, weights: 300, 400, 500, 600)
- **Текст:** тёмный, `#1a1a2e` для основного, `rgba(26,26,46,0.55)` для вспомогательного
- **Стиль заголовков:** нижний регистр в духе референса (glebfedorenko.com)

### 2.5 Cursor Glow

JS-эффект: радиальный градиент `rgba(160,100,255,0.12)` следует за курсором, `pointer-events: none`, `position: fixed`, `z-index: 9999`. Обновляется через `mousemove`.

### 2.6 Акцентный цвет

`#2563EB` (синий) — для ссылок, кнопок, столбчатых диаграмм.

## 4. Секции — детальное описание

### 4.1 Header (sticky)

- `position: sticky; top: 0; z-index: 100`
- Стеклянный фон: `rgba(255,255,255,0.7)` + `backdrop-filter: blur(20px)`
- Слева: имя "bogdan" или логотип (TBD — заглушка текстом)
- Справа: переключатель языка в glass-капсуле: `ru 🇷🇺` / `en 🇬🇧`
  - При выборе языка: JS меняет атрибут `data-lang` на `<html>`, все тексты переключаются через `data-ru` / `data-en` атрибуты на элементах (без перезагрузки)

### 4.2 Hero

Стеклянная карточка с macOS-chrome сверху.

**Содержимое:**
- Имя: `bogdan atrosenko` (крупно, ~48–56px)
- Роль: `ai entrepreneur / engineer` (~20px, вспомогательный цвет)
- Краткая фраза позиционирования (заглушка: `building AI-driven products`)
- Две CTA-кнопки рядом:
  - `→ telegram` (стеклянная кнопка с hover glow)
  - `→ email` (стеклянная кнопка с hover glow)

### 4.3 About + Media

Двухколоночный layout (на десктопе):

**Левая колонка — медиа:**
- Серый placeholder `aspect-ratio: 16/9`, `border-radius: 16px` — для looping MP4 видео (заменится на `<video autoplay loop muted playsinline>`)
- Стеклянная рамка вокруг

**Правая колонка — текст + bar charts:**
- 2–3 строки bio (заглушка)
- Блок bar charts (см. §4.3.1 ниже)

На мобильном: медиа сверху, текст + charts снизу.

#### 4.3.1 Bar Charts (Stats)

4 вертикальных столбика с цифрами. Анимация при попадании в viewport.

| Значение | Подпись |
|----------|---------|
| `2` | года опыта |
| `8+` | проектов |
| `3` | стека |
| `20` | лет |

**Высоты столбиков** (пропорционально максимуму = 20 = 100%):

| Значение | Высота |
|----------|--------|
| 20 | 100% |
| 8 | 40% |
| 3 | 15% |
| 2 | 10% |

**Реализация анимации:**
```js
// IntersectionObserver, threshold: 0.5
// При первом срабатывании → observer.unobserve()

// Столбики: transform: scaleY(0 → 1), transform-origin: bottom
// transition: transform 1.2s cubic-bezier(0.22, 1, 0.36, 1)

// Счётчик цифр: requestAnimationFrame, duration 1200ms
// easing: t => 1 - Math.pow(1 - t, 3)  // easeOutCubic
// Для "8+" — счётчик до 8, потом добавить "+"
```

**Цвет столбиков:** `rgba(37, 99, 235, 0.7)` (`#2563EB` с прозрачностью)

### 4.4 YouTube-блок

Стеклянная карточка. Превью YouTube-видео (thumbnail) — при клике открывает `youtube.com/...` в новой вкладке (`target="_blank"`).

Структура карточки:
- Заголовок секции: `watch / смотреть`
- Превью изображение (`<img>` с thumbnail URL или placeholder)
- Иконка Play поверх превью (SVG, красный YouTube play button)
- Подпись: название видео (заглушка)

URL видео и thumbnail — заглушки, заменятся позже.

### 4.5 Journey (карьерный путь)

Горизонтальная диаграмма-флоу в стеклянном стиле. Три узла с соединяющими стрелками.

```
[🧑‍💻 Разработчик] ──→ [⚙️ Инженер] ──→ [🚀 Предприниматель]
  Python · Backend      AI Agents           Свои проекты
  2 года найма          системная работа    сейчас
```

**Реализация:**
- Три стеклянных `div` с иконкой, заголовком, подзаголовком
- Между ними: SVG-стрелка или CSS `::after` со стрелкой
- Активный узел (Предприниматель) — чуть ярче обводка + glow
- Анимация: при попадании в viewport узлы появляются слева направо с задержкой (`animation-delay: 0s, 0.2s, 0.4s`)
- macOS-chrome декор на всём блоке-обёртке

На мобильном: вертикальный стек со стрелками вниз.

### 4.6 Achievements & Projects

Сетка 2×2 (десктоп) / 1×N (мобильный) стеклянных карточек.

Каждая карточка:
- Иконка / эмодзи
- Название (заглушка)
- Краткое описание (1–2 строки, заглушка)
- Теги стека в маленьких pill-бейджах
- Hover: обводка ярче, внешний glow `rgba(37,99,235,0.15)`

Статус: 3–4 заглушки, контент заполнится позже.

### 4.7 Contact

Финальная большая стеклянная карточка:
- Заголовок: `let's talk / поговорим`
- Две кнопки: `→ telegram` и `→ email`
- Вспомогательная строка (заглушка)

## 5. Двуязычность (ru / en)

**Механизм:** data-атрибуты на каждом текстовом элементе.

```html
<span data-ru="года опыта" data-en="years of experience"></span>
```

JS-функция `setLang(lang)` проходит по всем `[data-ru]` и ставит `textContent` из нужного атрибута. Текущий язык сохраняется в `localStorage`.

По умолчанию: `ru`. Переключатель в хедере меняет язык мгновенно.

## 6. Микро-взаимодействия

| Эффект | Реализация |
|--------|-----------|
| Cursor glow | `mousemove` → `position: fixed` радиальный градиент |
| Blob drift | CSS `@keyframes` translate + scale |
| Card hover | CSS transition на border + box-shadow |
| Bar chart appear | IntersectionObserver + CSS transform + JS counter |
| Journey appear | IntersectionObserver + CSS animation-delay |
| Lang switch | JS data-атрибуты, без перезагрузки |

## 7. Адаптивность

| Breakpoint | Поведение |
|-----------|-----------|

```
/
├── src/
│   ├── index.html        # единственная страница
│   ├── css/
│   │   ├── reset.css     # нормализация
│   │   ├── base.css      # переменные, типографика, фон, блобы
│   │   ├── glass.css     # переиспользуемые glass-компоненты
│   │   └── sections.css  # стили каждой секции
│   ├── js/
│   │   ├── cursor.js     # cursor glow
│   │   ├── lang.js       # переключатель языка
│   │   ├── charts.js     # bar chart анимация
│   │   └── journey.js    # journey анимация появления
│   └── assets/
│       └── video/        # placeholder для looping видео
├── dist/                 # выход сборки (gitignore)
├── package.json          # "build": "cp -r src/. dist/"
└── docs/
    └── superpowers/
        └── specs/
            └── 2026-06-22-personal-site-design.md
```

**Build script** (`package.json`):
```json
{
  "name": "bog-at-s-website",
  "version": "1.0.0",
  "scripts": {
    "build": "mkdir -p dist && cp -r src/. dist/"
  },
  "dependencies": {}
}
```
CI делает `npm ci` затем `npm run build` → `dist/`. Репозиторий должен содержать `package-lock.json` (генерируется `npm install` при пустых зависимостях). Зависимостей нет — Inter подключается через Google Fonts CDN в `<head>`.
