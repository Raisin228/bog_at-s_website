# Редизайн личного сайта-визитки — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Заменить glassmorphism/rainbow-blob дизайн на строгий editorial стиль с тёплым янтарным акцентом, единую систему токенов с сайтом агентства.

**Architecture:** Правки только в существующих файлах `src/css/` + `src/index.html` + `src/js/`. Новый файл `src/js/scroll.js` для единого fade-up наблюдателя. Никаких новых зависимостей.

**Tech Stack:** HTML / CSS / Vanilla JS, GitHub Pages

## Global Constraints

- Коммит-формат: `<Тег>: <сообщение на русском>`. Теги: `Feature`, `BugFix`, `Docs`, `ReDesign`, `CI`. Без `Co-Authored-By`.
- Никаких npm-зависимостей. Никаких фреймворков.
- Поддержка Safari: все `backdrop-filter` с `-webkit-` prefix.
- Все `data-ru` / `data-en` атрибуты на текстовых элементах сохранить.
- `lang.js`, `charts.js` — не трогать (logic не меняется, только CSS-переменные).
- Проверка после каждой задачи: `npm run build` должен завершаться без ошибок.

### Task 5: JS-анимации (`scroll.js` новый + `journey.js` обновление)

**Files:**
- Create: `src/js/scroll.js`
- Modify: `src/js/journey.js`

**Interfaces:**
- Consumes: классы `.fade-up`, `.fade-up.visible` из Task 3. Классы `.journey-node`, `.journey-node.visible` из Task 3.
- `scroll.js`: находит все `[data-fade-up]` или `.fade-up` элементы (кроме `.journey-node`), добавляет `.visible` при попадании в viewport.
- `journey.js`: наблюдает за `.journey-nodes`, добавляет `.visible` к `.journey-node` с задержкой `i * 0.15s`.

- [ ] **Шаг 1: Создать `src/js/scroll.js`**

```js
(function () {
  var elements = document.querySelectorAll('.fade-up:not(.journey-node)');
  if (!elements.length) return;

  var observer = new IntersectionObserver(function (entries) {
    entries.forEach(function (entry) {
      if (entry.isIntersecting) {
        entry.target.classList.add('visible');
        observer.unobserve(entry.target);
      }
    });
  }, { threshold: 0.15 });

  elements.forEach(function (el) {
    observer.observe(el);
  });
})();
```

- [ ] **Шаг 2: Заменить `src/js/journey.js` целиком**

```js
(function () {
  var nodesContainer = document.querySelector('.journey-nodes');
  if (!nodesContainer) return;

  var observer = new IntersectionObserver(function (entries) {
    entries.forEach(function (entry) {
      if (entry.isIntersecting) {
        var nodes = entry.target.querySelectorAll('.journey-node');
        nodes.forEach(function (node, i) {
          node.style.transitionDelay = (i * 0.15) + 's';
          node.classList.add('visible');
        });
        observer.unobserve(entry.target);
      }
    });
  }, { threshold: 0.3 });

  observer.observe(nodesContainer);
})();
```
